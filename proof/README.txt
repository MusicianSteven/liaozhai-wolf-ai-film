00_生成记录总览.txt
    33 条 HappyHorse 生成任务的自动汇总:每条含镜头号、模型/模式
    (happyhorse-1.1-r2v / -i2v)、生成参数(时长/分辨率/参考图数)、
    request_id、产出文件与大小。可与 GMI Cloud 后台任务记录逐条核对。

01_生成任务日志/(33 个 .log)
    每次生成的完整原始日志,文件名 = 产出镜头__任务ID.log。
    内容含:提交时间、模型与参数、request_id、逐次状态轮询
    (queued→processing→success)、下载记录。
    覆盖 S00-S15 全部镜头及探针版/迭代版(如 S07_v2/v3/v4,
    S09_probe~v6,S11_probe~v6,S12_v1~v3)。

02_生成参数存档/(28 个 .json)
    每镜生成的"真源"参数存档:完整 prompt、negative_prompt、
    duration、resolution、模式(r2v/i2v)、参考图/首帧说明。
    与日志中的 request_id 一一对应,全流程可复现。
